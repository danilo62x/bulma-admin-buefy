# Componentes do Buefy

## Layout
- **Layout & elements**: Componentes base para estrutura e organização visual da página.

## Actions
- **Button**: Botões clicáveis em várias cores, tamanhos e estados.
- **Dialog**: Caixas de diálogo modais para alertas, confirmações e solicitações de input.
- **Dropdown**: Menus suspensos acionados por clique ou hover.
- **Modal**: Janela de sobreposição (overlay) para exibir conteúdos diversos na frente da página.

## Data Display
- **Carousel**: Controle deslizante (slider) para apresentar imagens ou outros componentes.
- **Collapse**: Área de conteúdo que pode ser expandida ou retraída.
- **Icon**: Renderiza ícones em pacotes variados (suporte a MDI, FontAwesome, etc).
- **Image**: Imagem responsiva com recursos extras (como suporte a lazy loading e placeholders).
- **Loading**: Sobreposição de tela para indicar o carregamento de dados e bloqueio de interface.
- **Message**: Blocos de mensagens com suporte a diferentes cores e fechamento para realçar informações.
- **Notification**: Banners/notificações de estado que podem ser fechados pelo usuário.
- **Progress**: Barras de progresso nativas para demonstrar status de operações.
- **Skeleton**: Espaços reservados em formato de bloco (placeholder) mostrados enquanto um conteúdo é carregado.
- **Snackbar**: Pequenas mensagens temporárias de aviso que geralmente aparecem na parte inferior.
- **Table**: Tabelas de dados robustas com paginação, ordenação e filtragem.
- **Tag**: Pequenos rótulos/etiquetas usados para categorias ou marcações.
- **Toast**: Notificações efêmeras e não obstrutivas, similares aos snackbars, mas sem botão de ação obrigatório.
- **Tooltip**: Balão de dicas (texto pequeno) exibido ao se passar o cursor sobre um elemento.

## Form Controls
- **Autocomplete**: Campo de texto interativo que sugere opções de preenchimento.
- **Checkbox**: Caixas de seleção para entradas de opção binária (verdadeiro/falso).
- **Clockpicker**: Seletor de horas iterativo com formato de relógio analógico.
- **Colorpicker**: Ferramenta para selecionar cores e tons.
- **Datepicker**: Calendário dinâmico para a seleção de datas e intervalos.
- **Datetimepicker**: Componente combinado que agrupa a seleção de data e hora em um só lugar.
- **Field**: Invólucro essencial (wrapper) para controles de formulário, fornecendo agrupamento, rótulos e exibição de erros.
- **Input**: Campo nativo de entrada de textos.
- **Numberinput**: Campo de entrada numérico com controles de incremento e decremento.
- **Radio**: Botões circulares para opções mutuamente exclusivas de seleção única.
- **Rate**: Componente de avaliação visual dinâmico (por exemplo, estrelas de avaliação).
- **Select**: Campo de seleção tradicional do tipo dropdown.
- **Slider**: Barra de controle deslizante numérico (intervalos de valores).
- **Switch**: Botão de alternância estilo interruptor (liga/desliga).
- **Taginput**: Campo onde o texto inserido pelo usuário é convertido em etiquetas independentes (tags).
- **Timepicker**: Controle otimizado e focado apenas na seleção de horário.
- **Upload**: Botão de upload e zona para arrastar-e-soltar arquivos.

## Navigation
- **Breadcrumb**: Elemento de trilha de migalhas para mostrar a navegação hierárquica atual do usuário.
- **Menu**: Lista de navegação vertical tradicional (ótimo para uso em sidebars/painéis).
- **Navbar**: Barra principal de navegação superior, totalmente responsiva.
- **Pagination**: Controles numéricos e de "próximo/anterior" para navegar por longas listas ou tabelas separadas por páginas.
- **Sidebar**: Barra lateral do tipo "off-canvas" com sobreposição.
- **Steps**: Assistente passo-a-passo (wizard) dividindo tarefas complexas em etapas rastreáveis.
- **Tabs**: Componente em abas ou guias para transitar entre blocos de conteúdo na mesma visualização.
